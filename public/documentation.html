Movie API

This is a movie API, which contains movie description, genre, director, image URL.
The web application will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Features

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it is featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister.

API Endpoints

Buisness Logic URL HTTP Method Query Parameters Request body data format Response body data format
Get a list of all movies /movies GET - - A JSON object holding data about all the movies
Get a single movie by title /movies/:Title GET :Title - A JSON object holding data about a single movie
Get a list of all users /users GET - - A JSON object holding data about all the users
Get a single user by username /users/:Username GET :Username - A JSON object holding data about a single user
Get Genre by Name /genres/:genreName GET :genreName - A JSON object holding data about the genre of a movie
Get director data by name /directors/:directorName GET :directorName - A JSON object holding data about the data of a director
Allow new users to Register /users POST - We will expect JSON in this format { ID: Integer, Username: String, Password: String, Email: String, Birthday: Date } A JSON object holding data about the user that was added, structured like this { Username: String, Password: String, Email: String, Birthday: Date, "favorite movie" : [] } including an ID
Allow new users update their user info by username /users/:Username PUT :Username We will expect JSON in this format { Username: String (required), Password: String (required), Email: String (required), Birthday: Date } A JSON object holding data about the updated information
Adds movie to favorite list /users/:Username/favoritemovies/:MovieID POST :MovieID - A text message indicating whether the Movie was added
Delete movie from favorite list /users/:Username/favoritemovies/:MovieID DELETE :MovieID - A text message indicating whether the Movie was removed
Allow existing users to deregister by username /users/:Username DELETE :Username - A text message indicating whether the User has deregistered
